Hexo-搭建个人博客以及结合github、阿里云

发表于 2017-10-05 | 分类于 编程开发

Welcome to [L.Q 先生] 个人技术博客

准备工作

搭建一个本地简易博客

  • 首先安装hexo-cli

    npm install -g hexo-cli

  • 几行命令搞定建站 默认访问 locahost:4000

    hexo init blog_name
cd blog_name
npm  install 
npm  install hexo --save
npm  install hexo-server --save
hexo server

使用github Pages实现域名访问

  • 提前工作
    1. 将本机的公钥id_ssh.pub加入github账号中
      ssh-keygen -t rsa -C "xxxxx@xxxxx.com #打开id_ssh.pub文件复制，粘贴到如图所示；
      ssh -T git@github.com #测试是否添加key成功
    2. 创建项目(new repository)命名为”xxx.github.io” 无需创建自述文件readme.md
    3. 编辑blog_name的_config.yml文件 添加部署地址
      
deploy:
type: git
repo: git@github.com:xxx/xxx.github.io.git
branch: master
    4. 窗口安装插件” npm install hexo-deployer-git –save”，然后调用”hexo d -g” 即可部署成功到github page
    5. 若无法访问第一检查”xxx.github.io”项目是否打开了github page功能(打开xxx.github.io项目点击Settings打开页面查找Github Page进行打开);第二检查是否安装hexo-deployer-git插件，及其ssh_key是否添加成功。
